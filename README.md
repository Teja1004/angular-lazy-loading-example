## Quick start

```bash
# clone the repo
git clone https://github.com/abidakram01/angular-lazy-loading-example.git

# change directory
cd angular-lazy-loading-example

# install the repo with npm
npm install

# start the server
npm start

```
in your browser go to [http://localhost:4200](http://localhost:4200)
